# Ejemplos de facturas

Colocar aquí los PDFs de ejemplo que se usarán para desarrollar y probar parsers.

## Convención de nombres recomendada

```txt
electricidad/iberdrola/iberdrola_vallehermoso_2025_01.pdf
electricidad/naturgy_regulada/naturgy_uprose_2025_08.pdf
gas_natural/energia_xxi/energia_xxi_zarzaquemada_2025_02.pdf
```

Junto a cada PDF debe existir un `.expected.json` con la salida esperada del parser.
